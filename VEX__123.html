 Каталог Соревнования Загрузки

Наборы по робототехнике vex 123 позволяют изучать основы алгоритмов и программирования в начальной школе.

Благодаря входящему в состав наборов программатору и набору карточек на русском языке, учебный процесс может быть реализован без использования мобильных устройств И/или стационарных компьютерных систем. Кроме того, устройство может быть запрограммировано на выполнение алгоритмов посредством кнопок, расположенных на самом роботе.

Комплектация:

  • роботы разных цветов
  • программатор
  • карточки для программатора (50 в упаковке)
  • поле (4 пластины для основания и 8 бортиков)
  • устройство для зарядки с шестью портами usb
  • коробка для хранения программатора
  • коробка для хранения карточек
  • элементы для украшения (кольца)
  • батарейки aaa для программатора
  • отвертка
  • зарядка usb (3a)
